# Home Lab Infrastructure

## Overview
This project documents a self-hosted, high-availability lab environment designed to simulate enterprise IT infrastructure.

## Architecture
- 4-node Proxmox cluster
- Ceph distributed storage
- VLAN-segmented network
- Reverse proxy and DNS redundancy

## Key Features
- High availability virtualization
- Infrastructure automation (Terraform, Ansible)
- Secure network segmentation
- Redundant DNS architecture

## Technologies Used
- Proxmox
- Ceph
- Ubiquiti UniFi
- Docker
- BIND9
- Adguard home 
- NGINX Proxy Manager
- Cloudflare 
- Home Assistant 

## Goals
- Simulate enterprise IT environments
- Improve automation and reliability
- Practice troubleshooting and recovery scenarios
- Digital sovereignty
